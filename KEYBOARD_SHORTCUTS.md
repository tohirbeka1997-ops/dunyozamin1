# POS Terminal Keyboard Shortcuts - Quick Reference

## 🎯 Essential Shortcuts

| Key | Action | When to Use |
|-----|--------|-------------|
| **ENTER** | Add first search result to cart | After typing product name in search |
| **F2** | Open payment dialog | When ready to complete order |
| **F3** | Hold current order | When customer needs to wait |
| **ESC** | Close dialog or clear search | To cancel or go back |

## 🔢 Favorite Products

| Key | Action |
|-----|--------|
| **ALT+1** | Add favorite product #1 to cart |
| **ALT+2** | Add favorite product #2 to cart |
| **ALT+3** | Add favorite product #3 to cart |
| **ALT+4** | Add favorite product #4 to cart |
| **ALT+5** | Add favorite product #5 to cart |
| **ALT+6** | Add favorite product #6 to cart |
| **ALT+7** | Add favorite product #7 to cart |
| **ALT+8** | Add favorite product #8 to cart |

## 🛒 Cart Navigation

| Key | Action |
|-----|--------|
| **↑** (Up Arrow) | Select previous cart item |
| **↓** (Down Arrow) | Select next cart item |
| **+** (Plus) | Increase quantity of selected item |
| **-** (Minus) | Decrease quantity of selected item |

## 💡 Tips for Faster Workflow

### 1. Quick Product Entry
```
Type "apple" → Press ENTER → Product added!
```

### 2. Instant Favorites
```
Press ALT+1 → First favorite product added!
Press ALT+2 → Second favorite product added!
```

### 3. Fast Quantity Adjustment
```
Press ↓ to select item → Press + + + to add 3 more
```

### 4. Quick Payment
```
Press F2 → Select payment method → Complete!
```

### 5. Hold Order
```
Press F3 → Enter customer name → Hold!
```

## 🖱️ Mouse Alternatives

Instead of clicking, you can:
- Press **ENTER** instead of clicking "Add to Cart"
- Press **F2** instead of clicking "Process Payment"
- Press **F3** instead of clicking "Hold Order"
- Press **ESC** instead of clicking "Cancel" or "Close"

## 📱 Touch Screen Users

All shortcuts work with keyboard, but you can also:
- Tap favorite products to add them
- Tap cart items to select them
- Tap quantity/discount fields to open numpad
- Tap buttons for all actions

## 🎓 Training Recommendation

**Day 1:** Learn ENTER, F2, F3, ESC
**Day 2:** Learn ALT+1 to ALT+8 for favorites
**Day 3:** Learn arrow keys and +/- for cart navigation

After 3 days, most cashiers can work 50% faster!
