## NEW API FUNCTIONS

### src/db/api.ts
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ getDashboardAnalytics(startDate, endDate)
   - Returns all KPI metrics for date range
   - Queries: orders, order_items, products, sales_returns, purchase_orders
   - Safe for empty database (returns 0 instead of errors)

✅ getDailySalesData(startDate, endDate)
   - Returns daily sales breakdown for chart
   - Fills missing dates with 0 (no gaps in timeline)
   - Returns [] if no data

✅ getTopProducts(startDate, endDate, limit)
   - Returns top N products by sales amount
   - Aggregates quantity and total from order_items
   - Returns [] if no data
